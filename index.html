Overview:

Powered by smart contracts, this website provides a secure and trustless way to trade.




Sell:

To sell an item, you must create a contract. The Ethereum you send with the contract with be the value of the item. This will be used as collateral and will be returned to you after the transaction is complete.




Cancel:

You can cancel any contract you created that has not been bought.
